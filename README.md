El objetivo de este trabajo es diseñar e implementar una herramienta simple en Python que integre conceptos de Learning Analytics con elementos de criptografía y blockchain.

# Learning Analytics con Criptografía y Blockchain

Proyecto académico desarrollado en Python que integra:

- Learning Analytics
- Criptografía (AES/Fernet)
- Blockchain simulada mediante hashes SHA-256
- Visualización de datos educativos
- Predicción básica de rendimiento estudiantil

## Tecnologías

- Python
- Pandas
- Matplotlib
- Scikit-learn
- Cryptography
- Hashlib

## Funcionalidades

- Carga de datos desde CSV
- Encriptación de información sensible
- Verificación de integridad
- Detección de estudiantes en riesgo
- Visualización de métricas educativas

```bash
pip install -r requirements.txt
